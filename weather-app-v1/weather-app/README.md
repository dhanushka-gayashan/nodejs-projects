**_Technologies_**

    - JS
    - Nodejs
    - NPM

**_Installed NPM Libraries_**

	- request (or postman-requsst)
	- nodemon

**_3rd Party APIs_**

    - https://weatherstack.com/
    - https://www.mapbox.com/

**_Setup Instructions_**

    - Add "weatherstack" and "mapbox" api tokens into utils/api_tokens.js
    - npm install
    - nodemon src/app.js -e js
